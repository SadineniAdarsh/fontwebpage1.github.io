Process to include Different fonts in My HTML page:

  1. First, we should visit the Google Fonts website (fonts.google.com).
  2. Then we should search for the desired font using the search bar.
  3. Once we find the desired font, click on the "+" icon next to the font name to add it to your selection.
  4. Once you have selected all the fonts you need, click on the "Family Selected" bar at the bottom of the page.
  5. Lastly, we have to copy the provided HTML link element and paste it in the `head` section of my HTML document.
